# ①課題名
じゃんけんアプリ

## ②課題内容（どんな作品か）
- HTML・CSS・JavaScriptのみで作成した、割り勘計算アプリ「walica」のPC版。
- 「walica」の機能をPC一画面に集約。

## ③アプリのデプロイURL
なし

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID: なし
- PW: なし

## ⑤工夫した点・こだわった点
- 「walica」の機能をPC一画面に集約。
- 「walica」でできることはすべて網羅。計算ロジックも同様。
- 決算方法を画像で出力できるように機能追加。
- メンバー追加や立て替え追加をした際は、それと同時に動的に全画面が反映されるようにした。
- 立て替え一覧から、登録済みの立て替えを削除できるようにした。
- localstorageには、メンバー配列と立て替え履歴の配列の2つのみにし、jsonを用いた。
- 立て替え履歴の配列の中のオブジェクトの要素に、さらに配列を入れたい場合は、一度","区切りの文字列へと変換した。


## ⑥難しかった点・次回トライしたいこと（又は機能）
- 立て替え一覧から、登録済みの立て替えを修正できるようにするため、修正ボタンも用意したが、実装が間に合わなかった。修正画面は、追加のところを流用するのか、新しいポップアップウィンドウを出すのか、どちらの手段にしろ実装難易度が高かった。

## ⑦フリー項目（感想、シェアしたいこと等なんでも）
- 限られた時間の中でかなり生産性高く実装しきることができてうれしい。
