# ①課題名
じゃんけんアプリ

## ②課題内容（どんな作品か）
- コラショとしまじろうの二人のユーザー画面の切り替え可能なチャットアプリ(更新版)

## ③アプリのデプロイURL
デプロイしている場合はURLを記入（任意）

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID: なし
- PW: なし

## ⑤工夫した点・こだわった点
- コラショの端末画面と、しまじろうの端末画面の２つ用意し、テキストの左右寄せがユーザーの切り替えと同時に逆になるようにした。
- ２つ違うユーザーでウィンドウを開いて操作することで、同時に二人分の画面として操作可能。
- 画面の横幅を変えても、正常に表示されるように対応した。
- 送信時に送信音が鳴るようにした。
- テキストの吹き出しを実装した。
- LINEのように送信時間を吹き出しの横に表示されるようにした。
- 相手がチャットを見た時に、既読マークが表示されるようにした。
- LINEのようにスタンプも送信できるようにした。

## ⑥難しかった点・次回トライしたいこと（又は機能）
### 難しかった点
- ユーザーを切り替えた後に送信をすると、切り替えた回数分同じテキストが連続送信されてしまうことの修正。ユーザー切り替えのたびにこれまでのテキスト表示を一度消し、再表示にして表示処理が加算されないようにした。
- 既読を即時反映するために、firebaseが更新されたかどうかを監視する関数を導入したこと。
- リファクタリングに苦戦した。なるべく同じ処理やif分を減らすために、「?」と「:」を用いた条件分岐を使った。
### 次回トライしたいこと
- 今回は２人の登場人物しか想定してないので、自分なのか相手なのかで場合分けできたが、もっと複数人の個別チャットを実装するための、firebaseの保存する場所を増やすなど新たな試みをしてみたい。
